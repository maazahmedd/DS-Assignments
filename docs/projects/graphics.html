Graphics & UI Components

Interactive demonstrations of graphics rendering and UI event handling. Originally implemented in Java Swing, recreated here with HTML Canvas.

Target Drawing

Phone Keypad (Digicode)

Target Implementation

The target is drawn using the HTML Canvas API, mirroring the original Java implementation that used Graphics.fillOval().

// Draw concentric circles for (let i = 0; i < numRings; i++) { ctx.beginPath(); ctx.arc(cx, cy, radius, 0, Math.PI * 2); ctx.fill(); radius -= ringWidth; }

Keypad Implementation

The keypad demonstrates event handling, originally using Java's ActionListener interface. Here we use DOM event listeners.

// Event handling pattern btn.addEventListener('click', (e) => { const key = e.target.dataset.key; display.textContent += key; });

Original Java Code

These demos were originally part of an introduction to Java GUI programming using the Swing framework:

  • JFrame - Main window container
  • JPanel - Drawing surface and layout
  • JButton - Clickable buttons
  • Graphics - 2D rendering context
  • ActionListener - Event handling
