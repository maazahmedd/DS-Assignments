Maze Generator

Watch Prim's algorithm generate a perfect maze in real-time. A perfect maze has exactly one path between any two cells.

How It Works

This maze is generated using Prim's Algorithm, a greedy algorithm typically used for finding minimum spanning trees. When adapted for maze generation:

  1. Start with a grid where all cells have walls
  2. Pick a starting cell and mark it as visited
  3. Add all walls of the current cell to a wall list
  4. While there are walls in the list:
    • Pick a random wall
    • If one side is visited and the other isn't, remove the wall
    • Mark the new cell as visited and add its walls

Algorithm Complexity

Time Complexity: O(n log n)

Where n is the number of cells. Each wall is processed once, and the priority queue operations take log n time.

Space Complexity: O(n)

We store the grid of cells and a list of walls to process.
