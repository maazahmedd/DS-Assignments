How to Play

  1. Each player starts with 26 cards
  2. Click "Draw Card" to flip a card from each deck
  3. Higher card wins the round and scores a point
  4. Ace is highest (14), 2 is lowest
  5. Ties result in no points awarded
  6. Game ends when all cards are played

Card Rankings

A (14) > K (13) > Q (12) > J (11)

Implementation Details

This game uses the same singly linked list data structure from Assignment 2 to manage the deck of cards.

  • Deck class stores 52 Card objects
  • shuffle() uses random insertion/removal
  • Drawing cards uses removeFirst() - O(1)
  • Card comparison via isStrongerThan()
